# Gaia's Whisper  
# Sobre o Projeto
  * Este jogo foi desenvolvido usando unity, uma game engine com programação em C#.
  * O objetivo foi tentar criar uma experiência fluida, com desafio e mecânicas características do gênero metroidvania.
